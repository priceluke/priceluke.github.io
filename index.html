WebSAIF - Explicit Content Web Filter

August 4th, 2019

Over the Summer I created a Content Filter, aimed at parents, to block explicit content from appearing on web pages. We have all seen various quesitonable adverts appear so to keep this out of the view of children WebSAIF was created.

It is essencially a customisable Chrome extension that calls a Keras/Flask API that decides if the image is apropriate. The extension will then remove the image from the site if it deems it to be inapropriate. This was my first AI project and progressing the system from a binary clasification, either inapropriate or not, to a class based Convolutional neural network was a big challenge but allowed for far more customisation over what was filtered out on the client side.

I've kept the source code for this private for now but it may be avalible for download soon!
